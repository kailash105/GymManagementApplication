# Gym Management Application

A cross-platform React Native application for Gym Management with Role-Based Access Control (RBAC).

## Features
- **Three Roles**: Owner, Trainer, Member
- **Authentication**: Secure login/logout flow
- **Owner**: Manage revenue, members, trainers, and plans
- **Trainer**: View schedule and assigned members
- **Member**: View subscription and workout plans

## Tech Stack
- React Native (Expo)
- TypeScript
- React Navigation
- Axios (Mocked API)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Run on iOS/Android Simulator or Device.
